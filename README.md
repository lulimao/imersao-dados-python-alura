# imersao-dados-python-alura

Este projeto foi desenvolvido durante a Imersão Dados Python da Alura. Trata-se de uma aplicação interativa que permite explorar e visualizar dados salariais globais do mercado de dados, ajudando a entender tendências de remuneração por cargo, senioridade e localização geográfica.

Funcionalidades
O dashboard oferece uma visão macro e micro dos dados através de:

Métricas Principais (KPIs): Visualização rápida da média salarial, salário máximo, total de registros e cargo mais frequente na amostra selecionada.
Filtros Dinâmicos: Filtragem em tempo real por Ano (2020-2025), Senioridade (Júnior, Pleno, Sênior, Executivo), Tipo de Contrato e Tamanho da Empresa.

Gráficos Interativos:
Top 10 cargos com maiores médias salariais.
Histograma de distribuição salarial.
Proporção de tipos de trabalho (Remoto, Híbrido, Presencial).
Mapa-múndi (Choropleth) mostrando a média salarial para Cientistas de Dados por país.
Tabela de Dados: Acesso aos dados brutos filtrados para inspeção detalhada.

Tecnologias Utilizadas
O projeto foi construído utilizando o ecossistema Python:

Python: Linguagem base do projeto.
Pandas: Manipulação e tratamento da base de dados (versão 2.2.3).
Streamlit: Framework para criação da interface web interativa (versão 1.44.1).
Plotly Express: Criação de gráficos dinâmicos e mapas (versão 5.24.1)
